# Workspace Tools

Browser enhancements for the MapBiomas Alerta Workspace.

Workspace Tools is a lightweight Tampermonkey userscript that improves the audit interface by making better use of the available screen space.

## Features

- 1, 2 or 3 cards per row.
- Compact Before/After image layout.
- Automatic layout restoration after page updates.
- Persistent layout preferences.

## Installation

1. Install **Tampermonkey** in Google Chrome.
2. Open `chrome://extensions/`.
3. Enable **Developer mode**.
4. Open **Tampermonkey** and enable **Allow User Scripts**.
5. Install `workspace-tools.user.js`.
6. Reload the Workspace.
